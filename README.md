# [게임 기획서] 황혼의 기억 (Nightfall Memoris)

## 1. 프로젝트 개요
- **장르:** 전략 턴제 카드게임, 덱빌딩 로그라이트, 수집형 서브컬처
- **테마:** 러브크래프트 크툴루 신화 기반의 코즈믹 호러
- **핵심 서사:** 미지의 존재에 의해 세계와 인지가 지워지는 '침식' 현상. 세계의 기억을 보존하는 '재단'과, 침식 속에서 유일하게 살아남은 '주인공(기록자)'의 여정.

## 2. 캐릭터 및 유형 시스템
총 3인의 '기록자'로 파티를 구성하며, 각 기록자는 5가지 유형 중 하나를 가집니다.
1. **망각 (주인공 전용):** 실질적인 혼돈(공격) 유형에 가까운 특수 유형.
2. **혼돈 (공격형/딜러):** 적에게 피해를 입히는 데 특화.
3. **심연 (마법형/디버퍼):** 적의 능력치를 깎거나 상태 이상을 부여.
4. **이상 (지원형/서포터):** 아군 회복, 정신력 관리, 버프 부여.
5. **초차원 (만능형/밸런스):** 공수 밸런스가 잡힌 다재다능한 유형.
*※ 유형은 절대적이지 않으며, 캐릭터의 주된 특성을 나타내는 지표로 작용합니다.*

## 3. 핵심 전투 및 카드 시스템
- **덱 구성:** 각 기록자는 5장의 고유 카드를 가지며, 3인 파티 구성 시 총 15장의 카드가 하나의 덱으로 섞입니다.
- **턴 사이클:** - 매 턴 **5의 행동력(AP)**이 주어집니다.
  - 턴 시작 시 덱에서 **5장의 카드**를 랜덤하게 드로우합니다.
  - 행동력을 모두 소모하거나 카드를 낼 수 없을 때 턴을 종료합니다.
  - 턴 종료 시 손에 남은 카드는 모두 '버려진 카드' 더미로 이동합니다.
  - 드로우할 카드가 부족하면 '버려진 카드'를 다시 섞어 덱을 채웁니다.

## 4. 스테이지 진행 및 침식 게이지 (Erosion Gauge)
- **침식도 시스템:** 각 스테이지의 침식 게이지는 99%에서 시작합니다.
- **진행 방식:**
  - 일반 적 처치 시 침식 게이지가 **7% 감소**합니다.
  - 총 14번의 전투를 거치면 게이지가 1%가 되며, 이때 **스테이지 보스**가 등장합니다.
  - 보스 처치 시 침식 게이지가 0%가 되며 다음 스테이지로 넘어갑니다. (총 3스테이지)
- **이벤트 노드:** 전투 종료 후 현재 파티의 상태(정신력, 침식도, 파티원 유형 등)에 따라 랜덤한 선택형 이벤트가 발생합니다.
- **고정 휴식:** 7번째 전투와 14번째 전투(보스 직전) 직후에는 반드시 휴식/정비 이벤트가 발생합니다.

## 5. 진정한 보스 (최종전 기믹)
- **코즈믹 호러 엔드게임:** 3스테이지 클리어 시, 침식의 원흉인 '진정한 보스'와 조우합니다.
- **승리 조건 없음:** 이 보스는 처치할 수 없습니다. 플레이어 파티가 전멸할 때까지 **얼마나 오래 버티며 누적 피해를 입혔는지**가 최종 기록(Score)으로 남게 됩니다.

## 6. 정신력 (Sanity) 시스템
- **개념:** 스테이지 진행도인 '침식 게이지'와는 별개로 관리해야 하는 캐릭터 개별의 멘탈 수치입니다.
- **감소 조건:** 침식체를 목격하거나, 단일 전투의 턴이 길어질수록 지속적으로 감소합니다.
- **회복 수단:** 이벤트 선택지, 소모성 아이템, '이상' 유형 기록자의 특정 카드 등으로 회복할 수 있습니다.
- **광기 페널티:** 정신력이 일정 수치 이하로 낮아지면 스킬 카드의 효과가 기괴하게 변형되며, 바닥에 달하면 캐릭터가 기억을 잃고 전투에서 영구 이탈합니다.
