# [게임 기획서] 황혼의 기억 (Nightfall Memoris)

## 1. 프로젝트 개요
- **장르:** 전략 턴제 카드게임, 덱빌딩 로그라이트, 수집형 서브컬처
- **테마:** 러브크래프트 크툴루 신화 기반의 코즈믹 호러
- **핵심 서사:** 미지의 존재에 의해 세계와 사람들의 인지마저 지워지는 '침식' 현상. 세계의 기억을 보존하는 '재단'과, 침식 속에서 유일하게 존재가 지워지지 않고 살아남은 '주인공(기록자)'의 기억을 찾는 여정.

## 2. 세계관 기반 핵심 시스템 설정

### A. 정신력(Sanity) = 자아(기억) 유지력
- **설정 융합:** 전투 중 적(침식체)을 마주하거나 특수 패턴에 당하면 '정신력'이 감소합니다. 이는 곧 침식에 동화되어 '자신을 잊어가는 과정'을 의미합니다.
- **광기/침식 발동:** 정신력이 고갈되면 캐릭터는 자신의 존재를 망각하며, 카드가 왜곡되거나(기억의 혼선), 피아를 식별하지 못하거나, 심하면 전투에서 완전히 소멸(이탈)합니다.

### B. 파티 구성: '기록자'들의 합류 (수집형 서브컬처 요소)
- **주인공의 특이성:** 플레이어는 침식의 면역을 가진 유일한 존재인 '주인공' 하나로 여정을 시작합니다.
- **파티원 영입 (기록자 구출):** 침식된 스테이지(노드)를 탐험하며, 재단에서 파견되었거나 고립된 다른 매력적인 '기록자'들을 발견하여 3인 파티를 완성합니다.
- **캐릭터 수집/해금:** 로그라이트 반복 플레이를 통해 얻은 재화(혹은 기억의 파편)로 새로운 '기록자(캐릭터)'들을 영입 풀에 추가할 수 있습니다.

### C. 보상 및 성장: '만물의 기록' 복원
- **기억의 파편 (유물/Relic):** 보스를 처치하거나 이벤트를 통해 얻는 '유물'은 잃어버린 '세계의 기억 파편'입니다. 이를 수집할수록 파티는 잊혀진 과거의 지식(패시브 버프)을 되찾아 강해집니다.
- **로그라이트 반복의 이유:** 전투에서 패배하여 주인공이 기억을 잃고 쓰러져도, 재단이 주인공을 회수하여 다시 침식 구역으로 보냅니다. 이전 생애에서 찾아낸 일부 '기록'들은 영구적인 성장(메타 프로그레션)으로 남습니다.

## 3. 전투 시스템 요약 (고정 덱 전술)
- **AP(행동력) 공유:** 3인의 파티(기록자)가 하나의 행동력을 공유하며 최적의 효율을 냅니다.
- **적 행동 예고:** 침식체들의 다음 행동(공격, 디버프, 기억 소거 등)을 미리 파악하고, 각 캐릭터의 고유 스킬로 맞받아쳐야 합니다.
- **디버프 역이용:** 침식(디버프)을 단순한 페널티로 두지 않고, 이를 역이용하여 치명적인 공격을 가하는 조건부 전술이 포함됩니다.
